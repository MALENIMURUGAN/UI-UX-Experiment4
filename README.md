# UI-UX-Experiment4

## Aim:
To design wireframe diagrams for a grocery application using Balsamiq, including 6 essential screens: Welcome, Login/Signup, Home/Dashboard, Product Listing, Basket, and Checkout, focusing on layout structure and navigation flow without colors or graphics.

## Algorithm:
1. Identify 6 essential screens based on user journey from entry to purchase

2. Design each screen layout with headers, content areas, and navigation elements

3. Establish clear navigation paths between screens with consistent back buttons

4. Place UI elements strategically (forms, buttons, filters, product grids, totals)

5. Ensure uniform spacing and alignment across all screens

6. Export screens as PNG images for repository submission
